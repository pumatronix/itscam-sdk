# JPEG Metadata Visualizer

[Português (Brasil)](metadata-visualizer.md) | [English (US)](metadata-visualizer.en-US.md)

Use this page to inspect metadata embedded in JPEG images generated by ITSCAM cameras. The file is processed locally in your browser: the visualizer extracts each JPEG exposure from single-image or multipart files, shows the `key=value` pairs in a searchable/sortable table, and draws bounding boxes when it finds tags such as `CoordPlaca` and `ClassifierList`.

See [JPEG metadata (COM marker)](jpeg-metadata.en-US.md) for details about the format and the helpers available in the SDK.
